This project is a responsive Weather Forecast Web Application built using HTML, Tailwind CSS, and JavaScript.
It allows users to search for real-time weather information of any city and view a short-term forecast.

The application fetches live data from the OpenWeather API and displays it in a clean and user-friendly interface.



FEATURES :

 Search weather by city name
 Display current temperature, humidity, and wind speed
 5-day weather forecast
 Temperature unit toggle (°C / °F)
 Stores recently searched cities using Local Storage
 Responsive design (mobile, tablet, desktop)
